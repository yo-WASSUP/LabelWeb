# LabelWeb
一个基于网页的目标检测数据标注工具，方便一个团队合作标注目标检测数据集。不需要额外安装任何软件或者插件，只需要该电脑上有浏览器即可

# 重要注意事项!!!
1. 使用浏览器（最好是chrome浏览器）打开labelweb.html
2. **存储图像的文件夹需要与labelweb.html在同一路径!!!** 如果图片数量较多,建议使用软连接而不是简单的复制.
